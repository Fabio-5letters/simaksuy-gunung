## 🎨 Design System

### Color Palette
```
Primary Gradient:   Purple (#667eea) → Pink (#764ba2)
Success Gradient:   Green (#1e5631) → Teal (#3a9f5d)
Danger Gradient:    Pink (#f093fb) → Red (#f5576c)
Mountain Colors:    Green / Orange / Blue / Purple
```

### Typography
- **Font**: Segoe UI, Tahoma, Geneva (Modern & Clean)
- **Sizes**: Responsive - 0.75rem to 6xl
- **Weights**: 400, 500, 600, 700, 900 (Bold headers)

### Spacing
- **Consistent**: 4px, 8px, 12px, 16px, 24px units
- **Padding**: 1rem to 2rem pada cards
- **Gaps**: 1.5rem to 2rem antar elements

### Shadows & Depth
```
Card Shadow:  0 4px 6px rgba(0,0,0,0.1)
Hover Shadow: 0 10px 20px rgba(0,0,0,0.15)
Elevation:    Translate Y with shadow
```

### Animations
```
Fade In:      0.5s ease
Slide Down:   0.3s ease from -20px
Hover Lift:   translateY(-2px to -8px)
Scale:        1.05 - 1.1 pada hover
Pulse:        2s ease infinite
Spin:         1s linear infinite
```
